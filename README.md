# Sampler

**Abstract**\
The project consists of engineering an electronic musical sampler from scratch. The work combines hardware development with the implementation of dedicated software, creating a machine equipped with pressure-sensitive pads for managing audio samples, inspired by the history of hip-hop music. This experience, balancing hobby and education, has provided solid technical skills and is conceived to be constantly updated and perfected.
